Chieko Asakawa

Access to the internet is not easy for everyone, but Asakawa has found a way to allow internet usage for the blind. She is known for creating the browser plug-in IBM Home Page Reader, which is the most prevalent text to speech program available. Asakawa is a blind computer scientist who promotes tools that elevate the level of accessibility for disabled peoples using technology. In addition to IBM Home Page Reader, Asakawa also created a digital library for Braille documents and a word processor that converts Braille to speech in order to allow blind people more access to the internet as well as the world of technology.
